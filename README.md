# Горбилет — «Тайные агенты Петербурга»

Лендинг летней акции-вакансии Горбилета: набор двух «тайных агентов», которые
бесплатно посещают культурные события Петербурга и присылают честные отзывы.
Лендинг — промежуточный шаг между рекламой и переходом в сообщество MAX.

## Структура

```
gorbilet-agents-spb/
├── index.html              # страница лендинга
├── assets/
│   ├── css/
│   │   └── styles.css      # все стили
│   └── img/
│       ├── hero.jpg        # фото набережной (hero)
│       └── max-logo.png    # иконка мессенджера MAX
└── README.md
```

## Локальный запуск

Это статическая страница без сборки. Достаточно открыть `index.html` в браузере.
Для корректной загрузки шрифтов и относительных путей удобнее поднять локальный сервер:

```bash
python3 -m http.server 8000
# открыть http://localhost:8000
```

## Публикация на GitHub Pages

1. Запушить репозиторий на GitHub.
2. Settings → Pages → Source: ветка `main`, папка `/ (root)`.
3. Страница будет доступна по адресу `https://<username>.github.io/gorbilet-agents-spb/`.

## Что заменить перед запуском

- **Ссылки на кнопках** — сейчас `href="#"`. Подставить диплинк/ссылку на сообщество в MAX
  (две кнопки: в блоке «Как стать агентом» и в финальном блоке).
- **Логотип Горбилета** в шапке нарисован inline-SVG (приближение фирменного знака).
  Рекомендуется заменить на оригинальный SVG из брендбука.
- **Дата** «23 июня 2026» — обновить при необходимости (встречается в шаге 3 и в финальном блоке).
- **Аналитика** — при заливке внешнего трафика добавить в `<head>` счётчик
  (Яндекс.Метрика / VK Ads pixel) для ретаргетинга.

## Текущие данные

- Город: Санкт-Петербург
- Сроки: июнь — сентябрь
- Призовой фонд: 2 агента × 7 событий (5 выбирает Горбилет, 2 — агент с сайта)
- Точка приёма заявок: сообщество в MAX
